# training-kubebuilder

## how to build
* install Hugo
  * [https://gohugo.io/getting-started/installing](https://gohugo.io/getting-started/installing)
 
* fetch theme files
```console
git submodule update -i
```

* start the Hugo server on localhost

```console
hugo server -D
```

* build static pages

```console
hugo -D
```
